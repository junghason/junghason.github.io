# 과일 자동차 🚗🍎

만 6세 어린이를 위한 아주 쉬운 자동차 레이싱 게임입니다.
화면을 손가락으로 움직여 자동차를 조종하며, **과일은 모으고 장애물(돌멩이 등)은 피합니다.**
운전석에는 우리 아이 사진이 들어갑니다 🙂

## 특징
- 👆 손가락으로 화면을 움직이면 자동차가 따라와요 (드래그 조작, 가장 쉬운 방식)
- 🍎 과일을 먹으면 점수가 올라가요 (🍌🍓🍇🍊🍉…)
- 🪨 돌멩이·고깔은 살짝만 피하면 돼요 (하트 3개, 아주 관대한 충돌 판정)
- 🐢🐰🐆 속도 3단계 — 6세는 "천천히"를 추천해요
- 🔊 먹을 때 기분 좋은 소리, 부딪히면 알려주는 소리
- 🏆 게임이 끝나면 칭찬과 함께 모은 과일 개수를 보여줘요

## 파일
- `index.html` — 화면
- `style.css` — 스타일
- `game.js` — 게임 로직 (Canvas)
- `assets/driver.jpg` — 운전자(아이) 사진

## 로컬에서 실행
```bash
python3 -m http.server 8000
# 브라우저에서 http://localhost:8000/car-racing/ 접속
```

배포 후에는 아래 주소로 접속할 수 있어요:
```
https://junghason.github.io/car-racing/
```

> 태블릿/휴대폰에서는 크롬으로 위 주소를 연 뒤, 메뉴 → **홈 화면에 추가**를 하면
> 앱처럼 아이콘으로 실행할 수 있습니다.

## 운전자 사진 바꾸기
`assets/driver.jpg` 파일을 다른 사진으로 교체하면 운전자 얼굴이 바뀝니다.
얼굴이 가운데 오는 정사각형에 가까운 사진이 가장 예쁘게 나와요.
