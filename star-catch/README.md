# 별·하트 모으기 ⭐💖

초등학생(특히 여자아이)을 위한 간단하고 귀여운 "받기" 게임입니다.
캐릭터를 좌우로 움직여 하늘에서 콩콩 떨어지는 별·하트·사탕·꽃을 모아요.

## 플레이 방법

- **화면을 좌우로 움직이면** 캐릭터가 따라와요. (PC는 ← → 방향키)
- ⭐💖🍬🌸🍭🎀 를 받으면 점수가 올라가고, 캐릭터가 통통 점프해요.
- 🌈 **무지개**는 드물게 나와요 — 보너스 점수 + 하트를 한 칸 충전!
- 💣 **폭탄**은 받으면 안 돼요! 받으면 하트가 줄어요. (그냥 떨어뜨려 보내면 괜찮아요)
- 연속으로 받으면 **콤보 점수**가 쌓여요. (3콤보부터 보너스)
- 보물을 놓치면 하트가 하나 줄어요. 하트가 모두 없어지면 끝!
- 점수가 오를수록 점점 빨라지고, 보물이 더 자주(가끔 두 개씩) 떨어져요.

## 난이도

- 천천히 🐢 / 보통 🐰 / 빠르게 🦄 중에 골라요.

## 파일 구성

- `index.html` — 화면 구성(시작 / 게임 / 결과)
- `style.css` — 파스텔 톤 스타일 (모바일 대응)
- `game.js` — 게임 로직(받기, 콤보, 효과음, 배경음악)
- `assets/player.png` — 동그란 캐릭터 이미지

## 기술 메모

- 외부 라이브러리 없이 순수 HTML/CSS/Canvas + Web Audio로 동작해요.
- 배경음악은 Web Audio로 만든 짧은 멜로디 루프이며, 🎵 버튼으로 끄고 켤 수 있어요.
- 모바일(아이폰/안드로이드 태블릿)에서 전체화면으로 자연스럽게 동작하도록 처리했어요.
