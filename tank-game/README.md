# 풍선 탱크 🎈🚜

만 6세 어린이를 위한 아주 쉽고 평화로운 탱크 슈팅 게임입니다.
화면을 손가락으로 움직여 탱크를 조종하고, **하늘에서 내려오는 알록달록 풍선을 펑! 터뜨립니다.**
탱크 포탑에는 우리 아이 사진(꼬마 대장)이 빼꼼 들어갑니다 🙂

> 무서운 적이나 전투는 전혀 없어요. "풍선 터뜨리기"라는 밝고 안전한 컨셉이라
> 6살 아이가 즐기기에 딱 맞습니다.

## 특징
- 👆 손가락으로 화면을 움직이면 탱크가 따라와요 (드래그 조작, 가장 쉬운 방식)
- 👆 화면을 톡 누르면 솜사탕 포탄을 발사해요
- 🤖 **자동 발사**가 있어서, 가만히 좌우로 움직이기만 해도 풍선이 펑펑 터져요
- 🎈 풍선을 맞히면 펑! 색종이가 흩날리고 점수가 올라가요
- 💨 풍선을 놓치면 살짝 아쉬워요 (하트 3개, 아주 관대한 명중 판정)
- 🐢🐰🐆 속도 3단계 — 6세는 "천천히"를 추천해요
- 🔊 발사·터뜨리는 기분 좋은 소리, 경쾌한 배경음악
- 🏆 게임이 끝나면 칭찬과 함께 터뜨린 풍선 개수를 보여줘요

## 파일
- `index.html` — 화면
- `style.css` — 스타일
- `game.js` — 게임 로직 (Canvas)
- `assets/commander.jpg` — 꼬마 대장(아이) 사진

## 로컬에서 실행
```bash
python3 -m http.server 8000
# 브라우저에서 http://localhost:8000/tank-game/ 접속
```

배포 후에는 아래 주소로 접속할 수 있어요:
```
https://junghason.github.io/tank-game/
```

> 태블릿/휴대폰에서는 크롬으로 위 주소를 연 뒤, 메뉴 → **홈 화면에 추가**를 하면
> 앱처럼 아이콘으로 실행할 수 있습니다.

## 대장 사진 바꾸기
`assets/commander.jpg` 파일을 다른 사진으로 교체하면 탱크에 탄 얼굴이 바뀝니다.
얼굴이 가운데 오는 정사각형에 가까운 사진이 가장 예쁘게 나와요.
