# 대포 탱크 🎯🚜

만 6세 어린이를 위한 **앵그리버드 스타일** 발사각 조준 게임입니다.
탱크를 손가락으로 **쭈욱 당겼다 놓으면(슬링샷)**, 포탄이 포물선을 그리며 날아가
나무 상자로 쌓은 **장애물 탑을 와르르 무너뜨리고 풍선을 펑! 터뜨립니다.**
탱크 포탑에는 우리 아이 사진(꼬마 대장)이 빼꼼 들어갑니다 🙂

> 무서운 적이나 전투는 없어요. "조준해서 와르르 무너뜨리기"라는
> 밝고 안전한 컨셉이라 6살 아이가 즐기기에 딱 맞습니다.

## 노는 방법
1. 화면을 손가락으로 누른 채 **아래로 쭈욱 당겨요** (당긴 반대 방향으로 날아가요)
2. 하얀 **점선**을 보면서 발사각과 힘을 맞춰요
3. 손가락을 **떼면 발사!** 🚀
4. 풍선을 모두 터뜨리면 **스테이지 통과** 🏆

- 🐢🐰🐆 난이도(쉬워요/보통/어려워요)에 따라 포탄(발사 횟수)이 달라져요
- 🧱 상자 탑은 진짜로 와르르 무너져요 (간단한 물리)
- 🎈 풍선은 포탄에 맞거나 무너진 상자에 부딪히면 펑! 터져요
- ⏭️ 통과하면 다음 스테이지로, 포탄을 다 쓰면 다시 도전
- ⌨️ 데스크톱: ←→ 각도, ↑↓ 힘, 스페이스로 발사
- 🔊 발사·명중 효과음과 경쾌한 배경음악

## 파일
- `index.html` — 화면
- `style.css` — 스타일
- `game.js` — 게임 로직 (Canvas, 포물선 물리 + 상자 충돌)
- `assets/commander.jpg` — 꼬마 대장(아이) 사진

## 로컬에서 실행
```bash
python3 -m http.server 8000
# 브라우저에서 http://localhost:8000/tank-game/ 접속
```

배포 후 주소:
```
https://junghason.github.io/tank-game/
```

> 태블릿/휴대폰에서는 크롬으로 위 주소를 연 뒤, 메뉴 → **홈 화면에 추가**를 하면
> 앱처럼 아이콘으로 실행할 수 있습니다.
